## Prerequisites

### Numerical Analysis Libraries
```
sudo apt-get install libblas-dev liblapack-dev libatlas-base-dev gfortran
```
### Numpy, Scipy, Tensorflow (GPU version), Keras (1.2.0)
```
pip install numpy
pip install scipy
pip install tensorflow-gpu
pip install keras==1.2.0
```
